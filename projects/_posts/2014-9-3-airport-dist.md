JavaScript web application to calculate the distance between two airports and to plot the trip on Google maps. Auto-completes airport names and features all airports internationally.

## Powered by:

- Google Maps API
- SITA Airport API
- AirportCode API

## Technologies used:

HTML/CSS
JavaScript
AJAX
jQuery
Twitter Bootstrap
